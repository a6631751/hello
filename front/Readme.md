#前端

- 项目使用angularJs+Sass(compass)
- 采用yeoman + bower + grunt 集成开发环境

##调试需要的环境

1. 安装nodejs,ruby

2. 安装grunt-cli,bower,compass

> $ npm install -g grunt-cli

> $ npm install -g bower

> $ gem install compass

3. 在项目下执行:

> $ npm install

> $ bower install

##运行和单元测试
> $ grunt serve

> $ grunt test
